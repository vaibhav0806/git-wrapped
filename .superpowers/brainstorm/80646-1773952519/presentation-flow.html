How should the Wrapped presentation flow?

The core UX of the terminal experience

A

Spotify-Style Slides (Auto-Advancing)

One stat per screen, auto-advances every 3-4 seconds with smooth transitions. Spacebar to pause/resume, arrow keys to skip. Like watching a story unfold.

Pros

  • Most like actual Spotify Wrapped
  • Each stat gets its own dramatic reveal
  • Perfect for GIF export (plays like a video)
  • Simple mental model

Cons

  • Can't linger on interesting stats easily
  • Linear — no random access
B

Interactive TUI (Navigate Freely)

Full Bubble Tea app — navigate between stat "cards" with arrow keys, tab between sections. More like a dashboard you explore.

Pros

  • User controls the pace
  • Can revisit favorite stats
  • Feels like a polished app

Cons

  • Less "wrapped" magic — more like a report
  • Harder to export as GIF (no natural flow)
  • More complex to build
C

Hybrid: Auto-Play with Controls

Starts as auto-advancing slides (like A), but you can press arrow keys to take manual control at any time. Press 'a' to resume auto-play. Best of both worlds.

Pros

  • Dramatic reveals like Spotify Wrapped
  • User can pause and explore when curious
  • GIF export uses the auto-play path
  • Feels intentional and polished

Cons

  • Slightly more complex state management
